Welcome to GEO 455/555-Wemapping course. The webpage layout is designed to provide an example of a student's final webpage. Students are expected start with a simple webpage with a simple minimalist design and progressively improve it throughout the semester to showcase the skills developed during the course. Detailed information about the course is provided in the syllabus here.


Reading Materials

Assignment Description
  • Muehlenhaus, Ian (2013), Web Cartography: Map Design for Interactive and Mobile Devices 1st Edition, ISBN 9781439876220, CRC Press
    • Ch 3 - Map elements
    • Ch 4 - Map Composition and Layout
    • Ch 9 - Thematic Visualization
    • Ch 10 - Animation
    • Ch 12 - Web Map Production
  • "Introduction to Webmapping" by Michael Dorman (Web book)
    • Ch 1 - HTML
    • Ch 2 - CSS
    • Ch 3 - JavaScript Basics
    • Ch 4 - JavaScript interactivity
    • Ch 5 - Webservers
    • Ch 6 - Leaflet
    • Ch 7 - GeoJSON
    • Ch 12 - Client-side Processing
  • Lab 1: Introduction to HTML. The goal of this lab is to learn how to author and style an HTML document, create hyperlinks, email link, and add images. This lab is adapted from Ian Muehlenhaus at UWL.
  • Lab 2: Introduction to CSS. The goal of this lab is to learn how to create styled webpage layout which can be used to show different containers and embed webmap. This lab is adapted from Ian Muehlenhaus at UWL.
  • Lab 3: Webmap with point markers. The goal of this lab is to learn how to create a map with list of point locations and create customize pop-up.
  • Lab 4: Layer grouping and WMS. The goal of this lab is to learn how to group layers using Leaflet JS package to create customized menu, use multiple basemap, use custom popups with images, fetch data from a Web Map Service (WMS).
  • Lab 5: Adding JavaScript interactivity. The goal of this lab is to learn how to add real time data via open API, add JS interactivity such as navigation buttons plugins, clickable basemaps, and add custom plugins and widgets.
  • Lab 6: Choropleth mapping with GeoJSON. The goal of this lab is to learn how to work with existing geojson data structure, create choropleth map, convert shapefile to json file. This lab is adapted from Ting Liu at North Eastern Illinois University.
  • Lab 7: Thematic Mapping. The goal of this lab is to learn how to create different thematic maps, such as proportional circles, heatmap, and cluster map using Leaflet JS package. This lab is based on undergraduate research conducted by Sydney Graff (2020) under the supervision of Prof. Chaudhuri.
  • Lab 8: jQuery and ESRI Leaflet. The goal of this lab is to learn how to add interactivity using jQuery on a webpage, and use ESRI Leaflet package to fetch custom data from ArcGIS online
  • Lab 9: Flowmap. The goal of this lab is to learn how to use Leaflet JS plugin to create a flow map using American Community Survey Commuting Flows data.
  • Lab 10: Introduction to Turf.JS. The goal of this lab is to learn how to work add basic geoprocessing in a webmap using TurfJS.
